# Grant Summary

## Project Summary

Orina Protocol is a settlement layer for bilateral commerce. ATP gives marketplaces, wallets, AI-assisted workflows, and commerce applications a reusable protocol for escrowed, time-bounded, dispute-aware transactions that reach one final settlement outcome.

## Problem

Many commerce applications still rely on platform discretion, manual escrow handling, or informal coordination to determine whether a multi-step transaction has completed. This creates ambiguous final state, unresolved escrow, inconsistent records, and duplicated settlement logic across applications.

## Solution

ATP standardizes settlement outcomes for bilateral transactions:

- agreement and transaction initiation,
- escrowed payment handling,
- seller and buyer settlement states,
- timeout handling,
- dispute boundary,
- release or refund outcome,
- final settlement record.

## Ecosystem Impact

Ecosystems can use Orina as settlement infrastructure rather than funding another isolated marketplace. Applications can integrate a common transaction finality layer while keeping their own user experience, discovery model, asset category, and business workflow.

## Funding Use

Grant funding helps move Orina from public testnet evidence toward production readiness through independent review, network-specific integration, public documentation, partner onboarding, and mainnet preparation.
